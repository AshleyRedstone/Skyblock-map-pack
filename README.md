# Skyblock Map Pack

Backup packs for Ashley's bedrock skyblock map

## Compiled downloads

- `dist/Skyblock_Map_Pack.mcaddon` - behavior pack and main resource pack
- `dist/Skyblock_Classic_RP.mcpack` - optional pre-1.14 texture override

## Pack folders

- `behavior_packs/Skyblock_BP` - behavior pack
- `resource_packs/Skyblock_RP` - main resource pack
- `resource_packs/Skyblock_Classic_RP` - optional pre-1.14 textures
