# Skyblock Map Pack

Generated Minecraft Bedrock packs for compressed Skyblock blocks.

## Pack folders

- `behavior_packs/Skyblock_BP` - behavior pack
- `resource_packs/Skyblock_RP` - main resource pack
- `resource_packs/Skyblock_Classic_RP` - optional pre-1.14 texture override

Activate **Skyblock Classic** above **Skyblock** in the resource-pack order when using the classic textures.

The classic pack is intentionally sparse. It contains only generated overrides whose source textures exist in `bases/old`.
